# Myriad Mobile (React Native with Expo)

This is the mobile app for Project Myriad, built with React Native and Expo.

## Getting Started

1. Install dependencies:
   ```bash
   cd mobile
   npm install
   ```
2. Start the Expo development server:
   ```bash
   npm start
   ```
3. Scan the QR code with the Expo Go app (iOS/Android) or run on web with:
   ```bash
   npm run web
   ```

## Features
- Cross-platform: Android, iOS, and Web
- Ready for Web3 integration
